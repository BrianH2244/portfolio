My Problem Solving Method


1. Define the problem.

  • Make sure I ask the right question, that is specific enough to the problem.

2. Verify the problem exists.

  • Has the problem occured on more than one computer? Can I replicate it?
  • Gather basic information about the right circumstances where the error occured.
  • What browser was being used? What version of browser? What platform?

3. Break down the problem.

  • Read the problem completely, twice. Solve the problem manually with sets of sample data.
  • Optimize the manual steps. Write the manual steps as comments or pseudo-code.
  • Replace the comments or pseudo-code with real code. Optimize the real code.

4. Use tools to gather more information about the problem.

  • Chrome Javascript Console, Firebug, W3C Validation, Fiddler.

5. Check test data. It may be faulty data, not faulty code or algorithms.

6. Back-up work before making any changes.

7. Review past experiences versus the present situation. Have I solved this before?

8. Search for people who've been in similar situations and how they solved the problem.

  • “Google search” defined question. Look at existing solutions e.g. design patterns.

9. Make one change at a time - test it. Keep the scope of changes small and contained.

10. Ask for help. Explaining the problem can help you better understand it, allowing them to ask a question or point out a detail you may not have realized.

11. Act as if the problem has already been solved. Trace what follows back to the solution.

12. Take a break. Back off and rest, to solve subconsiously, and get new perspectives.

13. Check the problem for inherent contradictions or conflicting requirements.

  • Make compromises if need be, and change the requirements of the problem.

14. If completely lost, bang out ideas and code, and improve them as they go.

15. Imperfect Solutions and Compromises.

  • If time sensitive, scale back features to when things were functioning perfectly and send live.
  • Employ the “best solution right now” vs the “best solution” in general.

Most importantly, think positive, persist and don't give in to frustration!
